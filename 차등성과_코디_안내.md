## 1. 한눈에 보는 차등 성과 제도

> **3일 동안의 성과를 모아서, 매번 새로운 등급을 받는 제도예요.**

- ✅ 회차 단위 = **3일**
- ✅ 등급 = **5단계** (A · B · C · D · H)
- ✅ 평가 기준 = **취소까지 반영한 계약 성과**
- ✅ 좋은 성과 = 위 등급(A) · 낮은 성과 = 아래 등급(D/H)
- ✅ 매 회차마다 **새롭게 줄세움** — 한 번 떨어졌다고 끝이 아님

## 2. 5개 등급 체계

| 등급 | 색상 | 의미 | 한 줄 |
|---|---|---|---|
| **A** | 🟨 골드 | 상위권 | "이 회차 잘했어요!" |
| **B** | 🟩 그린 | 중위권 | "안정적으로 일하고 있어요" |
| **C** | 🟧 오렌지 | 하위권 | "조금만 더 분발이 필요해요" |
| **D** | 🟦 블루 | 무성과 | "이번 회차 실질 성과가 0 이하예요" |
| **H** | ⬛ 그레이 | 페널티 | "D 3회 누적 후 4번째도 D면 진입" |

### 직관적으로 보면

```
   잘함  ────────────────  못함
   A  →  B  →  C  →  D  →  H
   (위)              (아래)
```

> 💡 **A는 가장 좋은 등급, H는 페널티 등급입니다.** 매 회차마다 본인의 성과로 새 등급을 받습니다.

## 3. 무엇으로 평가받나요? — 핵심 지표

### 평가 지표는 단 하나, **"취소반영 계약률"** 입니다.

```
취소반영 계약수  =  실계약수  −  취소수
취소반영 계약률  =  취소반영 계약수  ÷  DB수
```

### 쉽게 말하면

> **"받은 DB 중에서, 취소까지 빼고 남은 진짜 계약이 몇 % 인가?"**

### 예시

| 코디 | 받은 DB | 실계약 | 취소 | 취소반영 계약수 | 취소반영 계약률 |
|---|---|---|---|---|---|
| 김OO | 30건 | 9건 | 1건 | 8건 | 26.7% |
| 이OO | 30건 | 6건 | 0건 | 6건 | 20.0% |
| 박OO | 30건 | 5건 | 5건 | 0건 | 0% (→ **D**) |

> 📌 **계약을 많이 해도 취소가 많이 나면 점수가 깎입니다.** 진짜 살아남은 계약이 중요합니다.

### 취소 카운트 규칙 (참고)

- 반품일이 이번 회차 안에 있어야 함
- 매칭되는 계약의 오더일이 **달력상 3개월 이내**일 때만 카운트
- 즉, 3개월 보다 오래된 계약의 취소는 평가에 영향 없음

## 4. 평가는 며칠 단위로 이뤄지나요?

### 회차 = **3일**

```
[1일차] → [2일차] → [3일차]  =  1회차

       ↓
  이 3일 동안의 성과를 모아서 등급 산정
```

- 매 회차가 끝나면 **그 회차의 데이터만 가지고** 새로 등급 산정
- 이전 회차에 어떤 등급이었든, **이번 회차 성과로 새 등급 부여**
- 단, 일부 규칙(H 진입, H 탈출 등)에는 직전 회차 이력이 영향을 줍니다

## 5. 등급은 이렇게 정해져요 (산정 방식)

### 1단계 — D 등급 자동 분류

> **취소반영 계약수가 0건 이하인 사람은 자동으로 D**

- 0건 = 한 건도 못 했거나, 계약했는데 다 취소된 경우
- 마이너스 = 취소가 실계약보다 많았던 경우
- 둘 다 D로 묶입니다

> 💡 또한 **회차 내 근태이슈가 2건 이상**이면 C에 있더라도 D로 강등됩니다 (자세한 내용 §12).

### 2단계 — 나머지 인원을 A/B/C로 1/3씩 분할

> **D를 제외한 나머지 인원**을 취소반영 계약률 높은 순으로 줄세워서:

```
상위 1/3  →  A 등급
중위 1/3  →  B 등급
하위 1/3  →  C 등급
```

### 즉, 정리하면

```
[코디 전원]
   │
   ├─→ 취소반영 계약수 ≤ 0  →  D
   │
   └─→ 취소반영 계약수 > 0  →  취소반영 계약률 줄세우기
                              ├ 상위 1/3 → A
                              ├ 중위 1/3 → B
                              └ 하위 1/3 → C
```

> 📌 **같은 성과인 사람은 같은 등급**으로 묶입니다. 동률에서 갈리지 않아요.

## 6. 좋은 등급으로 가려면? — 행동 지침

### ① 취소를 줄이세요

- 계약을 10건 했어도 5건이 취소되면 취소반영 계약수는 5건
- 무리한 마감보다 **지속될 계약**이 점수에 더 도움

### ② 취소반영 계약수 1건 이상은 반드시 확보

- 0건 이하는 무조건 D
- 1건이라도 살아남으면 C 이상에 들어갈 수 있음

### ③ 출근 일수를 채우세요

- 회차 3일 중에 결근/지각이 **2번 이상이면 등급 한 단계 강등**
- A였어도 → B / B였어도 → C / **C였어도 → D** 로 떨어집니다

### ④ B/C에서 1등 노리기

- B 1위는 **A로 승급**
- C 1위는 **B로 승급**
- 등급 안에서 1등하면 한 단계 위로 올라갈 수 있음 (자세한 내용 §7)

---

## 7. 한 번 더 기회가 있어요 — 프로모션(승급)

### 매 회차마다 자동으로 적용되는 보너스 룰

```
B 등급 1위(실계약수 기준)  →  A 등급으로 승급
C 등급 1위(실계약수 기준)  →  B 등급으로 승급
```

### 왜 이 규칙이 있을까요?

> 같은 등급 안에서도 **노력한 사람은 한 단계 위로** 올라갈 수 있게 하는 보너스입니다.

### 1위 판정 기준

1. **실계약수**가 가장 많은 사람
2. 동률이면 **취소반영 계약률**이 더 높은 사람

### 단, D 등급은 프로모션 대상이 아닙니다

- D는 "성과 0 이하"가 정의이므로, D 안에서의 1위는 의미가 없어 승급되지 않습니다
- D에서 벗어나려면 **다음 회차에 취소반영 계약수 1건 이상** 만들어야 합니다

## 8. H등급은 어떻게 되나요?

### H 진입 — 만성 무성과 페널티

> **D 등급을 3회 받은 상태에서 다음(4번째) 회차도 D면 H 등급으로 강등됩니다.**

```
회차1: D  (D 1번 누적)         ┐
회차2: D  (D 2번 누적)         │   D 누적 중
회차3: D  (D 3번 누적, ⚠ 위험) ┘   ← 여기까지는 아직 D
   │
   ▼
회차4: D                          →  H 강등 (4번째 D 받는 순간)
```

### 화면 표시로 미리 알 수 있어요

엑셀에 본인의 **D 누적 횟수**가 색깔 배지로 표시됩니다.

| D 누적 횟수 | 표시 색 | 의미 |
|---|---|---|
| 0 | 회색 | 정상 (이번 회차 D 아님) |
| 1 | 🟨 노랑 | D 1번 |
| 2 | 🟧 주황 | D 2번 |
| **3** | 🟥 **빨강** | **위험! 다음 회차도 D면 H!** |

### H 탈출 — 한 번에 회복할 수 있어요

> **H 등급에서 이번 회차에 실계약 2건 이상**을 만들면 → 바로 **C 등급으로 복귀**됩니다.

- D 누적 횟수도 처음(0)부터 다시 시작 → 다시 D 3회 누적 후 4번째도 D를 받아야 H 진입
- 즉, **H는 영구가 아닙니다.** 한 번의 좋은 회차로 빠져나올 수 있어요

## 9. 이런 경우엔 어떻게 되나요? — 케이스별 안내

### 케이스 1 — 회차 3일 동안 정당사유로 모두 쉬었어요

> **직전 회차 등급 그대로 유지됩니다.**
- 평가가 사실상 불가능한 회차이므로, 직전 회차 성과로 등급 인정
- 인정되는 정당사유: 연차 / 휴가 / 공가 / 경조 / 대체휴무 / 오전 반차 / 오후 반차

### 케이스 2 — 1~2일 정당사유 + 나머지 출근

> **정상 평가됩니다.** (출근한 날의 성과로 평가)
- 정당사유 회차로 인정되지 않음
- 출근일에 받은 DB · 만든 계약으로 평가

### 케이스 3 — 정당사유 1일 + 결근 2일

> **정상 평가 + 근태이슈 강등 적용** (한 단계 ↓)
- 결근이 섞이면 정당사유 회차로 인정되지 않음
- 결근 2일 → 근태이슈 2건 → 한 단계 강등

### 케이스 4 — 회차 3일 모두 결근

> **정상 평가 + 근태 강등** (한 단계 ↓, 자동 D 가능성 매우 높음)

### 케이스 5 — 입사한 지 한 달 됐는데 D가 나왔어요

> **신입 보호로 B 등급 부여됩니다.** (자세한 내용 §10)

### 케이스 6 — 계약 5건 했는데 4건이 취소됐어요

> **취소반영 계약수 = 1건** → C 또는 그 위 등급
- 취소가 많아도 1건이라도 살아남으면 D는 면합니다

### 케이스 7 — DB를 한 건도 못 받았어요

> **자동 D 분류** — DB가 없으면 평가가 불가능하므로 D
- 다음 회차에 분배를 받아 정상 평가 가능

## 10. 신입 보호 제도

### 입사 +60일 이내 코디는 보호받습니다

> **자동 산정 결과가 C / D / H라면 → B 등급으로 상향**

- 입사 후 60일 동안은 **하향 방지**
- 단, 성적이 좋아서 자동으로 A 또는 B를 받았다면 그대로 유지 (A도 가능)
- D 누적 횟수도 0으로 초기화됩니다

### 표시

- 비고 컬럼에 **`신규입사자`** 라고 표시됩니다
- B 등급 안에서 일반 B와 시각적으로 구분됨 (B 등급 최하단에 정렬)

> 💡 **60일이 지나면 보호가 풀립니다.** 그 이후엔 모든 코디와 동일한 기준으로 평가됩니다.

## 11. 정당 사유 (휴가/연차) 처리

### 정당사유로 인정되는 사유

```
연차 · 휴가 · 공가 · 경조 · 대체휴무 · 오전 반차 · 오후 반차
```

### 정당사유 회차로 처리되려면?

> **회차 3일 모두가 정당사유**로 채워져야 합니다.

| 케이스 | 처리 |
|---|---|
| 3일 모두 연차/휴가 | ✅ **정당사유 회차** → 직전 등급 유지 |
| 1~2일 정당사유 + 나머지 출근 | ❌ **정상 평가** (출근일 성과로 등급 산정) |
| 1일 정당사유 + 2일 결근 | ❌ **정상 평가 + 근태 강등** |

> 📌 결근/지각이 1일이라도 섞이면 정당사유 회차로 인정되지 않습니다 (근태이슈 우선).

### 정당사유 회차의 효과

- 등급: **직전 회차 그대로**
- D 누적 횟수: **그대로** (늘지도 줄지도 않음)
- 다른 모든 룰(H 탈출, 근태 강등, 프로모션) **적용 안 됨**

## 12. 근태와 등급의 관계

### 근태이슈로 카운트되는 것

```
결근 · 오전 결근 · 오후 결근 · 조퇴 ·
지각 · 지각 10분 · 지각 20분 · 지각 20분 초과
```

### 강등 규칙

| 회차 내 근태이슈 | 처리 |
|---|---|
| 0건 | 영향 없음 |
| 1건 | 표시만 됨 (강등 X) — `근태 1` 노란 뱃지 |
| **2건 이상** | **한 단계 강등** ⚠ — `⚠ 근태 N` 빨강 뱃지 |

### 강등의 방향

```
A  →  B    (한 단계 ↓)
B  →  C    (한 단계 ↓)
C  →  D    (한 단계 ↓) ★
D  →  D    (유지, 더 떨어지지 않음)
H  →  H    (유지)
```

> ⚠️ **C에서도 근태이슈 2건이면 D로 떨어집니다.** 계약을 1건 이상 했더라도 결근/지각이 누적되면 D 등급을 받을 수 있으니 출근 관리가 매우 중요합니다.

> 📌 **근태이슈는 회차 단위로 독립 카운트** 됩니다. 다음 회차로 누적되지 않아요.

## 13. 자주 묻는 질문

### Q1. 매 회차마다 등급이 달라질 수 있나요?

A. **네, 매번 새로 산정됩니다.** 이번 회차에 D였어도 다음 회차에 잘하면 A까지도 가능합니다. (단, H 진입/탈출만 직전 이력의 영향을 받습니다.)

### Q2. 한 번 H가 되면 영원히 H인가요?

A. **아니요.** H 등급에서 이번 회차에 **실계약 2건 이상**을 만들면 즉시 **C 등급으로 복귀**됩니다. D 누적 횟수도 처음(0)부터 다시 시작됩니다.

### Q3. C와 D의 차이가 정확히 뭐예요?

A.
- **C** = 취소반영 계약수가 **1건 이상**인데, A/B에 못 든 사람 (=하위 1/3)
- **D** = 취소반영 계약수가 **0건 이하**인 사람 (계약을 못 했거나 다 취소됨)

### Q4. 등급은 어디서 확인할 수 있나요?

A. 영업관리팀에서 매 회차 종료 후 **차등 성과 엑셀 파일**을 공유드립니다. 본인 등급, 받은 DB수, 실계약수, 취소수, 취소반영 계약률, D 누적 횟수가 모두 표시됩니다.

### Q5. 회차 중에 입사한 신입은요?

A. 입사 후 **60일 동안은 신입 보호**가 적용됩니다. C/D/H로 자동 산정되어도 **B 등급으로 상향**되어 안정적으로 적응할 수 있습니다.

### Q6. 회차 3일 모두 휴가를 쓰면 등급이 안 떨어지나요?

A. **네, 직전 회차 등급이 그대로 유지**됩니다. 평가가 불가능한 회차이므로 페널티가 없습니다. 단, **결근이 1일이라도 섞이면 적용되지 않으니** 주의해주세요.

### Q7. C에서 1등을 했는데 왜 B가 아니고 그대로 C에요?

A. 다음 중 하나에 해당했을 가능성이 높습니다:
- 정당사유 회차였음 (직전 회차 등급 그대로 유지)
- 직전 회차 H였다가 이번에 C로 복귀(H 탈출자)

이 경우 **프로모션 풀에서 제외**되어 차순위에게 승급이 적용됩니다.

### Q8. 등급이 잘못 매겨진 것 같아요. 어떻게 하나요?

A. 영업관리팀 담당자에게 문의해주세요. 데이터 확인 후 필요시 수동으로 조정 가능합니다.
